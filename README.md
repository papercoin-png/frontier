# 🪐 Voidfarer - Orbital View

A stunning, immersive orbital view for a Telegram Mini App. Features procedurally generated pixel-art planets with atmospheric glow, starfields, and a spaceship HUD — all running smoothly on mobile devices.

## ✨ Features

- 🌍 **Procedural Planets** – 20+ planet types with infinite variations
- 🎨 **Pixel-Art Style** – Modern retro aesthetic inspired by reference images
- 🔄 **Smooth Rotation** – 60fps planet animation
- ✦ **Parallax Starfield** – Deep space background with depth
- 📱 **Mobile-First** – Optimized for Telegram Mini Apps
- 🚀 **Immersive HUD** – Ship instruments and planet data
- 🎮 **Interactive** – SCAN, LAND, and NEXT buttons (demo-ready)

## 🚀 Quick Start

### Option 1: Run Directly (No Installation)

1. Clone this repository
2. Open `index.html` in your browser
3. That's it! The orbital view runs immediately

### Option 2: Run with React (For Development)

```bash
# Clone the repository
git clone https://github.com/yourusername/voidfarer-orbital-view.git
cd voidfarer-orbital-view

# Install dependencies
npm install

# Start development server
npm start
